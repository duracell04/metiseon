# Metiseon Demo (Landing)

Static marketing/demo site built with Next.js, Tailwind CSS, and TypeScript.

## Stack
- Next.js (App Router)
- Tailwind CSS
- TypeScript
- pnpm for package management

## Getting Started
```bash
cd landing
pnpm install
pnpm dev
```

## Build
```bash
pnpm build
pnpm start
```

## Notes
- UI components come from the existing shadcn-style library in `src/components/ui`.
- Public assets live in `public/`.
